# 🤖 Robotic Arena Tracker (Система комп'ютерного зору та трекінгу)

Високопродуктивна серверна система комп'ютерного зору, написана на **Rust**, для відстеження роботизованої платформи та перешкод на випробувальному стенді. Система обробляє відеопотік з верхньої веб-камери, обчислює координати об'єктів та відправляє телеметрію на робота через протокол UDP у реальному часі.

## ✨ Ключові можливості

* **⚡ Висока продуктивність:** Написано на Rust з використанням біндингів OpenCV. Обробка кадрів та відправка телеметрії відбувається без затримок (до 30+ FPS).
* **👁️ Гібридний трекінг силуетів:** Алгоритм стійкий до перепадів освітлення, тіней та відблисків. Використовується математична морфологія (Dilation/Erosion) та побітове злиття масок (Bitwise OR) для склеювання глянцевого пластику та матових деталей у єдиний монолітний об'єкт.
* **🛡️ Динамічний радар перешкод:** Сервер автоматично розраховує Евклідову відстань від центру робота до всіх перешкод. Об'єкти в радіусі $\le 20$ см класифікуються як критичні.
* **🎛️ Live Calibration Tool:** Інструмент інтерактивного калібрування прямо під час роботи. Дозволяє "на льоту" підлаштовувати фільтри HSV (яскравість та насиченість) під матрицю будь-якої камери за допомогою клавіатури.
* **📡 UDP-Телеметрія:** Максимально швидка передача координат по мережі без затримок на "рукостискання" (handshakes).

## 🛠 Технологічний стек

* **Мова:** Rust
* **Комп'ютерний зір:** OpenCV 4.x
* **Захоплення відео (Linux):** V4L2 (Video for Linux 2)
* **Мережа:** UDP Sockets
